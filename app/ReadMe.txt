# Rowing Weather App

An Android app designed to provide rowers with accurate weather information tailored for rowing conditions. Stay informed about wind, temperature, and water conditions to plan your rowing sessions safely and efficiently.

## Features
Import from version control using this link: https://github.com/amethyst-oj/Rowing-app.git
In android studio and simply run the code.

Dependencies
Libraries used:
	Jackson faster xml
	androidx appcompat
	google material components
	constraint layout
	glide
	jsoup
	gson
	mpandroid chart
	firebase crashlytics
	google maps library
Also relies on www.weatherapi.com , google maps api
